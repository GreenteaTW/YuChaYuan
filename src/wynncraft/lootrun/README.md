## 什麼是 Lootrun

Lootrun是遊戲後期主要的賺錢管道之一，可以從中獲取有價值的物品
`白話文 : 可以拿到: 常見素材、綠寶石、Mythic、Shiny Mythic、部分裝備、特殊裝備`

::: tip 第一次接觸Lootrun

請先到 `Cinfras` 的 [-102, 36, -4814]
找到 `Lootrun Chief` 進行對話完成 Lootrun 的註冊

:::
